📋 FIREBASE API KEY ISSUE - QUICK SUMMARY
==========================================

❌ ERROR: auth/api-key-not-valid

🔍 WHAT'S WRONG?
Your Firebase API key in .env.local is either:
- Invalid for this project
- Restricted to certain APIs
- From a different Firebase app

🔧 QUICK FIX (5 MINUTES):

1️⃣  Go to: https://console.firebase.google.com/project/clubwiz-auth/settings/general
    
2️⃣  Scroll to "Your apps" section
    
3️⃣  Click on your Web app (or add one if missing)
    
4️⃣  Copy the apiKey from the config shown
    
5️⃣  Update .env.local with the new key:
    NEXT_PUBLIC_FIREBASE_API_KEY=YOUR_NEW_KEY_HERE
    
6️⃣  In Firebase Console > Authentication > Sign-in method:
    ✅ Enable "Phone" 
    ✅ Add "localhost" to authorized domains
    
7️⃣  Restart your app: npm run dev
    
8️⃣  Test: http://localhost:3001/auth/mobile

✅ DONE! Your phone auth should now work!

📚 See FIREBASE_API_KEY_FIX.md for detailed guide
